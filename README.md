# PulseHIS 🏥

A comprehensive **Hospital Information System** built to streamline healthcare operations and bring digital efficiency to clinics and hospitals.

![Dashboard Preview](./screenshots/dashboard.png)

## ✨ Features

### 📊 Admin Dashboard
- Real-time system overview with key metrics
- Total users, active doctors, appointments, and revenue tracking
- Quick action buttons for core operations
- Time-range filtering (7 days, 30 days, quarter)
- Appointment trends & doctor workload distribution charts
- PDF export functionality

### 👥 Patient Management
- Patient directory with search & filtering
- Patient registration with detailed profiles
- CSV export capability
- Gender-based filtering

### 👨‍⚕️ Doctor Management
- Doctor directory with specialization search
- Add/register new doctors
- Specialization-based filtering

### 📅 Appointment System
- Appointment listing with date-range filters
- Book new appointments
- Interactive calendar view
- Search by patient, doctor, or reason

### 💰 Financial Dashboard
- Revenue tracking & analysis
- Outstanding, paid, and unpaid invoice tracking
- Revenue trend charts (14-day view)
- Billing & invoice management
- Invoice creation module

### 🎨 UI/UX
- Dark mode & Light mode toggle
- Responsive sidebar navigation
- Clean, modern card-based design
- Role-based access (Staff Member/Admin)
- Search functionality across all modules

## 🛠️ Tech Stack

- **Frontend:** React (Vite)
- **Backend:** Node.js / Express
- **Database:** MongoDB
- **State Management:** Context API / Redux
- **Styling:** CSS3 / Tailwind CSS
- **Charts:** Chart.js / Recharts
